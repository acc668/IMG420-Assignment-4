# Assignment-4

A simple 2D platformer built with Godot 4.4 and C#.

## How to Run

1. Open Godot 4.4
2. Import the project
3. Press F5 or click Run

## Controls

- **Arrow Keys**: Move left (A) / right (D)
- **Space**: Jump

## Features

- Player movement with gravity and jumping
- Enemy AI that chases the player
- Collectible pickups that increase score
- Health system with enemy damage
- Particle effects for pickups and hits
- Game restarts when health reaches 0

## Project Structure

- `Scenes/` - All game scenes
- `Scripts/` - C# game scripts
- `Assets/` - Sprites and tilesets

## Requirements

- Godot 4.4 with .NET support
- .NET SDK 7.0+
